Open Access

저자의 비용 부담, 이용자의 무료 접근, 시공간을 초월한 상시적 접근, 저자의 저작권 보유 등 4대 원칙을 강조한 정보 공유 체제

협정기관들이 발행한 Open Access 데이터 및 개인 생산 데이터를 등록하여 공유 및 활동이 가능한 국가적 오픈 액세스망 구축

Data Library

플랫폼 등록 데이터를 대출하여 이용자가 직접 분석 생산하고 이 데이터를 다시 제공받는 데이터 선순환 체계를 지향하는 ‘데이터 서비스 도서관’ 서비스 연계

파이프라인 시대에서
플랫폼 시대로의 전환

  • 지식정보 수집·공유 강화

    관리자 중심의 정보지식 ‘포털’을 벗어나, 사용자가 관리 주체가 되어 정보자원을 활용하는 차세대 정보 생태계 구축

  • 지식정보 수집·공유 강화

    데이터 융합 기능을 제공함으로써 창작·가공한 자료의 자유로운 업로드·다운로드를 통해 지식정보의 수집과 공동 활용 활성화

  • 협의회 내실화

    5,900여 개의 회원기관에 달하는 한국학술정보협의회의 외연적 성장과 더불어 협정기관 간 실효성 있는 커뮤니티를 이룸
